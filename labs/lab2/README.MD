## Лабораторная работа 2

Написать скрипт для составления спортивного календаря.
На вход подается список команд с городами, в которых они находятся.

1. Игры проводятся по пятницам, субботам и воскресеньям, начало игр в 12:00, 15:00, 18:00.
2. Одновременно нельзя проводить более 2 игр.
3. Игры надо расположить как можно более равномерно в указанном пользователем диапазоне.
4. Все входные данные нужно валидировать
5. Календарь выводится в текстом формате в указанный пользователем файл.
6. Запуск осуществляется так: ruby build_calendar.rb teams.txt 01.08.2026 01.06.2027 calendar.txt
Указан 
7. Форматирование календаря на ваше усмотрение, но даты должны выглядеть хорошо.

## Темы, которые используются в работе:
1. Запуск скриптов Ruby
2. Чтение и запись файлов
3. Работа со строками
4. Работа с датами

Показать код преподавателю практики, запустить программу, ответить на вопросы.
